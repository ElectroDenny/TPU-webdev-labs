# TPU-webdev-labs
Репозиторий содержит лабораторные работы по базовому курсу веб-разработки.

## Содержание
В данном репозитории представлены следующие лабораторные работы:

### task1
* `catalog.html`: каталог тортов с таблицей, содержащей информацию о тортах и их стоимости.
* `questionnaire.html`: анкета с информацией о себе, включая фотографию, биографию, семью, увлечения и мечты.

### task2
* `form.html`: анкета для веб-разработчика с полями для ввода регистрационного имени, даты рождения, пароля, специализации, пола и навыков.

## Описание
Данные лабораторные работы выполнены в рамках базового курса веб-разработки и демонстрируют основные навыки в создании веб-страниц с использованием HTML и CSS.

## Использование
Чтобы просмотреть лабораторные работы, просто откройте соответствующие файлы в браузере.
